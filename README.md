## Run React.JS

Go to the project directory for client React.JS



```bash
  cd client
```

Install dependencies

```bash
  yarn install
```
Start React.JS 

```bash
  yarn start
```

## Run the server

Go to the project directory for server Node.JS

```bash
  cd server
```

Install dependencies

```bash
  yarn install
```

Start Node.JS server

```bash
  yarn dev (use nodemon) 
  or
  yarn start
```
